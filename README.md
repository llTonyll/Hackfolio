### Prerequisites
* [NodeJS](https://nodejs.org/en/)
* [MongoDB](https://www.mongodb.com/)
* [Brunch](http://brunch.io/)

### Installation

```bash
git clone https://github.com/jbehuet/blog-app.git   
cd blog-app
npm i
```

### Development

`npm run dev`

### Production

`npm start`
